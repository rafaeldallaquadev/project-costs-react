# Costs

Aplicação React para gerenciamento de projetos e serviços.

O usuário pode:

- criar projetos
- definir orçamento
- adicionar/remover serviços
- acompanhar custos do projeto

## Tecnologias

- React
- React Router
- JavaScript
- CSS
- JSON Server

## Funcionalidades

- CRUD de projetos
- Controle de orçamento
- Cadastro de serviços
- Atualização dinâmica de custos

## Como rodar

```bash
npm install
npm run dev
npm run backend
```
